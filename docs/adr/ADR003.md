# ADR-003: Biblioteca para conexão com o banco de dados.

**Status:** Proposto |  **Data:** [2026-05-28]  |  **Autores:** [Carlos Tomasi, Gustavo Antonius, Leonardo Latzke e Renan Apolináro]

## Contexto

Testar e escolher a melhor biblioteca para conexão com o banco de dados (Pyodbc ou SQLAlchemy).

A solução será responsável por: Escolher a melhor biblioteca para conexão com o banco de dados.

## Consequências

(+) Quando a conexão já está aberta, a consulta chega a ser 90% mais rápida.

(-) A primeira execução da Azure Function pode demorar um pouco mais, pois a biblioteca precisa carregar seus módulos e mapear o banco antes de rodar a query.

## Alternativas rejeitadas

Pyodbc
Apresentou resultados piores em multiplas consultas consecutivas.
